#  Student Grade Tracker

##  Overview
This project implements a **Student Grade Tracker** using Python fundamentals.  
It processes student data, performs analysis, and generates structured outputs using core programming concepts.

---

##  Concepts Used
- Variables & Data Types  
- Lists and Dictionaries  
- Loops (`for`, `while`)  
- Conditional Statements (`if-else`)  
- String Manipulation  
- Functions  
- Input Handling  

---

##  Tasks Implemented

### 🔹 Task 1 — Data Parsing & Profile Cleaning
- Clean raw student data:
  - Remove extra spaces
  - Convert names to Title Case
  - Convert roll numbers to integers
  - Convert marks from string to list
- Validate student names (alphabet check)
- Print formatted student profile cards
- Display name in uppercase and lowercase for a specific student

---

### 🔹 Task 2 — Marks Analysis
- Display subject-wise marks with grades:
  - A+, A, B, C, F
- Calculate:
  - Total marks
  - Average marks
  - Highest scoring subject
  - Lowest scoring subject
- Implement a **while loop** for adding new subjects
- Handle invalid input safely

---

### 🔹 Task 3 — Class Performance Summary
- Calculate average marks for each student
- Assign Pass/Fail status
- Display formatted class report
- Compute:
  - Number of students passed/failed
  - Class topper
  - Overall class average

---

### 🔹 Task 4 — String Manipulation Utility
- Clean and process a paragraph:
  - Strip whitespace
  - Convert to Title Case
  - Count occurrences of "python"
  - Replace text with formatted output
- Split text into sentences
- Print numbered sentences
